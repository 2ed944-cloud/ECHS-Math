Inverse function notation

Same relationship

\[ g=f^{-1}\quad\Longleftrightarrow\quad f(g(x))=x\quad\text{and}\quad g(f(x))=x \] If \(f(a)=b\), then \(f^{-1}(b)=a\).

Do not confuse

\[ f^{-1}(x)\ne \frac{1}{f(x)} \] The superscript \(-1\) means inverse function, not reciprocal.
Complete the statements:
1. If \(f(7)=-2\), then \(f^{-1}(-2)=\_\).
2. If \((3,5)\) is on \(y=f(x)\), then \(\_\) is on \(y=f^{-1}(x)\).
Explain why a function that is not one-to-one cannot have an inverse function on its full domain.
Answers
1. \(7\). 2. \((5,3)\). A non-one-to-one function repeats an output, so the inverse would assign one input to more than one output.

Graph connection

The graphs of \(f\) and \(f^{-1}\) are reflections across \(y=x\).

If the tangent slope to \(f\) at \((a,b)\) is \(m\ne0\), then the tangent slope to \(f^{-1}\) at \((b,a)\) is \(\frac1m\).
If \(f(3)=8\) and \(f'(3)=5\), what do you know about \(f^{-1}\)?
Answers
\((f^{-1})(8)=3\) and \((f^{-1})'(8)=\frac15\).

Derivative of an inverse function

Formula

\[ (f^{-1})'(x)=\frac{1}{f'(f^{-1}(x))} \] Equivalently, if \(f(a)=b\), then \[ (f^{-1})'(b)=\frac{1}{f'(a)}. \]
Find \((f^{-1})'(25)\) if \(f(x)=3x^3+1\).
Find \((f^{-1})'(2)\) if \(f(x)=x^3+2x-1\).
Answers
For \(f(x)=3x^3+1\): solve \(3a^3+1=25\Rightarrow a=2\). Then \(f'(2)=36\), so \((f^{-1})'(25)=\frac1{36}\).
For \(f(x)=x^3+2x-1\): solve \(a^3+2a-1=2\Rightarrow a=1\). Then \(f'(1)=5\), so \((f^{-1})'(2)=\frac15\).

Function-based inverse derivatives

  1. Find \((f^{-1})'(7)\) for \(f(x)=x^3-1\).
    Answers
    Solve \(a^3-1=7\Rightarrow a=2\). Since \(f'(2)=12\), \((f^{-1})'(7)=\frac1{12}\).
  2. Find \((f^{-1})'(7)\) for \(f(x)=4/x+2\).
    Answers
    Solve \(4/a+2=7\Rightarrow a=\frac45\). \(f'(x)=-4/x^2\), so \(f'(4/5)=-\frac{25}{4}\). Thus \((f^{-1})'(7)=-\frac4{25}\).
  3. Find \((f^{-1})'(1/2)\) for \(f(x)=\sin x\) on \([-\pi/2,\pi/2]\).
    Answers
    \(a=\pi/6\), \(f'(a)=\cos(\pi/6)=\sqrt3/2\). So \((f^{-1})'(1/2)=\frac{2}{\sqrt3}\).
  4. Find \((f^{-1})'(1)\) for \(f(x)=\cos(2x)\) on \([0,\pi/2]\).
    Answers
    \(\cos(2a)=1\Rightarrow a=0\). \(f'(0)=-2\sin0=0\), so the inverse derivative is not defined at \(1\).

When the inverse derivative does not exist

If \(f'(a)=0\) and \(f(a)=b\), then \((f^{-1})'(b)\) does not exist as a finite derivative.
Find \((f^{-1})'(0)\) for \(f(x)=x^3\). Interpret geometrically.
Why is \((f^{-1})'(b)=\frac{1}{f'(a)}\) impossible when \(f'(a)=0\)?
Answers
For \(f(x)=x^3\), \(f^{-1}(x)=\sqrt[3]{x}\), which has a vertical tangent at \(x=0\). The formula would require \(1/0\), so no finite derivative exists.

Inverse trigonometric derivatives

Core formulas with chain rule

\[ \frac{d}{dx}\arcsin u=\frac{u'}{\sqrt{1-u^2}},\qquad \frac{d}{dx}\arccos u=-\frac{u'}{\sqrt{1-u^2}} \] \[ \frac{d}{dx}\arctan u=\frac{u'}{1+u^2},\qquad \frac{d}{dx}\operatorname{arccot} u=-\frac{u'}{1+u^2} \] \[ \frac{d}{dx}\operatorname{arcsec} u=\frac{u'}{|u|\sqrt{u^2-1}},\qquad \frac{d}{dx}\operatorname{arccsc} u=-\frac{u'}{|u|\sqrt{u^2-1}} \]
Use \(u\)-substitution mentally: identify the inside function first, then multiply by its derivative.

Graph and domain reminders

\[ \arcsin x:\ -1\le x\le1,\quad -\frac\pi2\le y\le\frac\pi2 \] \[ \arccos x:\ -1\le x\le1,\quad 0\le y\le\pi \] \[ \arctan x:\ -\infty<x<\infty,\quad -\frac\pi2<y<\frac\pi2 \]

Derive the formulas

  1. Given \(y=\arcsin x\), use implicit differentiation to derive \(\frac{d}{dx}\arcsin x\).
    Answers
    \(\sin y=x\Rightarrow \cos y\,y'=1\). Since \(\cos y=\sqrt{1-x^2}\) on the principal range, \(y'=\frac1{\sqrt{1-x^2}}\).
  2. Given \(y=\arccos x\), derive \(\frac{d}{dx}\arccos x\).
    Answers
    \(\cos y=x\Rightarrow -\sin y\,y'=1\). Since \(\sin y=\sqrt{1-x^2}\), \(y'=-\frac1{\sqrt{1-x^2}}\).
  3. Given \(y=\arctan x\), derive \(\frac{d}{dx}\arctan x\).
    Answers
    \(\tan y=x\Rightarrow \sec^2 y\,y'=1\). Since \(\sec^2 y=1+\tan^2 y=1+x^2\), \(y'=\frac1{1+x^2}\).

Basic inverse trig derivatives

  1. Differentiate \(y=\arcsin(3x)\).
    Answers
    \(y'=\frac{3}{\sqrt{1-9x^2}}\).
  2. Differentiate \(y=\arctan(2x^2)\).
    Answers
    \(y'=\frac{4x}{1+4x^4}\).
  3. Differentiate \(y=\operatorname{arcsec}(5x)\).
    Answers
    \(y'=\frac{5}{|5x|\sqrt{25x^2-1}}=\frac{1}{|x|\sqrt{25x^2-1}}\), where defined.
  4. Differentiate \(y=\arccos(2x^5)\).
    Answers
    \(y'=-\frac{10x^4}{\sqrt{1-4x^{10}}}\).

Inverse trig with products and exponentials

  1. Differentiate \(y=(2x)\arcsin x\).
    Answers
    \(y'=2\arcsin x+\frac{2x}{\sqrt{1-x^2}}\).
  2. Differentiate \(y=x^2\arctan(3x)\).
    Answers
    \(y'=2x\arctan(3x)+\frac{3x^2}{1+9x^2}\).
  3. Differentiate \(y=e^{\arccos x}\).
    Answers
    \(y'=-\frac{e^{\arccos x}}{\sqrt{1-x^2}}\).
  4. Differentiate \(y=\ln(\arcsin x)\).
    Answers
    \(y'=\frac{1}{\arcsin x\sqrt{1-x^2}}\), where defined.

Tangent lines with inverse trig

  1. Find the tangent line to \(y=\arcsin(2x)\) when \(x=\frac14\).
    Answers
    At \(x=1/4\), \(y=\arcsin(1/2)=\pi/6\). \(y'=\frac{2}{\sqrt{1-4x^2}}\), so slope \(\frac{4}{\sqrt3}\). Tangent: \(y-\frac\pi6=\frac{4}{\sqrt3}\left(x-\frac14\right)\).
  2. Find the tangent line to \(y=\arccos(3x)\) when \(x=-\frac{\sqrt3}{6}\).
    Answers
    \(3x=-\sqrt3/2\), so \(y=5\pi/6\). \(y'=-\frac{3}{\sqrt{1-9x^2}}=-6\). Tangent: \(y-\frac{5\pi}{6}=-6\left(x+\frac{\sqrt3}{6}\right)\).
  3. Find the tangent line to \(y=\arctan(\sqrt{x})\) when \(x=3\).
    Answers
    \(y=\arctan(\sqrt3)=\pi/3\). \(y'=\frac{1}{1+x}\cdot\frac{1}{2\sqrt{x}}=\frac{1}{2\sqrt{x}(1+x)}\). At \(x=3\), slope \(\frac{1}{8\sqrt3}\).

Inverse function challenge

  1. Let \(f(x)=x^5+x\). Without finding \(f^{-1}\), compute \((f^{-1})'(2)\).
    Answers
    Solve \(a^5+a=2\Rightarrow a=1\). \(f'(1)=6\), so \((f^{-1})'(2)=\frac16\).

Inverse trig: domain traps

  1. Find the domain of \(y=\arcsin(2x-1)\).
    Answers
    Require \(-1\le2x-1\le1\), so \(0\le x\le1\).
  2. Find the domain of \(y=\arccos(x^2-2)\).
    Answers
    Require \(-1\le x^2-2\le1\Rightarrow 1\le x^2\le3\). Thus \(x\in[-\sqrt3,-1]\cup[1,\sqrt3]\).
  3. Find the derivative of \(y=\arcsin(2x-1)\) and state where the derivative is defined.
    Answers
    \(y'=\frac{2}{\sqrt{1-(2x-1)^2}}\), defined for \(0<x<1\).

Inverse trig: secant and cosecant

  1. Differentiate \(y=\operatorname{arcsec}(3\ln x)\).
    Answers
    \(y'=\frac{3/x}{|3\ln x|\sqrt{9(\ln x)^2-1}}\), where \(|3\ln x|>1\).
  2. Differentiate \(y=\operatorname{arccsc}(2x^2+1)\).
    Answers
    \(y'=-\frac{4x}{|2x^2+1|\sqrt{(2x^2+1)^2-1}}\). Since \(2x^2+1\ge1\), this simplifies to \(-\frac{4x}{(2x^2+1)\sqrt{(2x^2+1)^2-1}}\), where defined.
  3. Differentiate \(y=\operatorname{arcsec}(5x^2)\).
    Answers
    \(y'=\frac{10x}{|5x^2|\sqrt{25x^4-1}}=\frac{2}{|x|\sqrt{25x^4-1}}\), where \(|x|>1/\sqrt5\).

Harder inverse function reasoning

  1. Let \(f\) be one-to-one and differentiable. Suppose \(f^{-1}(4)=2\) and \((f^{-1})'(4)=\frac13\). Find \(f(2)\) and \(f'(2)\).
    Answers
    \(f^{-1}(4)=2\Rightarrow f(2)=4\). Since \((f^{-1})'(4)=1/f'(2)=1/3\), \(f'(2)=3\).
  2. Let \(f(x)=x^3+ax+1\). Find \(a\) if \((f^{-1})'(1)=\frac15\).
    Answers
    Since \(f(0)=1\), \((f^{-1})'(1)=1/f'(0)=1/a\). Thus \(a=5\).
  3. Let \(f(x)=x^3-3x+2\). Is \((f^{-1})'(2)\) guaranteed to be unique? Explain.
    Answers
    No. \(f(0)=2\) and \(f(\sqrt3)=2\) as well? The function is not one-to-one on all \(\mathbb R\); an inverse must be restricted before the derivative is meaningful.

  2. Differentiate \(y=\arcsin(\arctan x)\).

Challenge: endpoint behavior

  1. Does \(\frac{d}{dx}\arcsin x\) exist at \(x=1\)? Explain.
    Answers
    No finite derivative. The derivative formula \(1/\sqrt{1-x^2}\) blows up as \(x\to1^-\); the graph has a vertical tangent at the endpoint.
  2. Does \(\frac{d}{dx}\arccos x\) exist at \(x=-1\)? Explain.
    Answers
    No finite derivative; \(-1/\sqrt{1-x^2}\) is unbounded as \(x\to-1^+\).
  3. For \(y=\arcsin(2x-1)\), where does \(y'\) fail inside the domain?
    Answers
    Domain \([0,1]\). The derivative fails at the endpoints \(x=0\) and \(x=1\); it exists on \((0,1)\).
