Implicit Differentiation

3.2

Finding derivatives when a curve is not solved for \(y\)

\[x^2+y^2=16\qquad \frac{dy}{dx}\text{ is found by differentiating both sides with respect to }x.\]
Differentiate implicit equations Use the chain rule on every \(y\)-expression Find slopes, tangent lines, normal lines, and horizontal or vertical tangents
Solutions
  1. Differentiate both sides with respect to \(x\). Every derivative of a \(y\)-expression must include \(y'\).
  2. Move all terms containing \(y'\) to one side, factor \(y'\), and divide.
  3. For tangent-line questions, substitute the point only after finding \(y'\), then use \(y-y_1=m(x-x_1)\).
  4. For horizontal tangents use numerator \(=0\) with denominator \(\ne0\); for vertical tangents use denominator \(=0\) with numerator \(\ne0\).

Opening decision

Classify each equation as explicit or implicit. Then choose whether ordinary rules alone are enough or whether implicit differentiation is needed.

  2. \(x^2+y^2=25\)
  3. \(y=\sin(x^2)\)
  4. \(\ln(2xy^2)=4x\)
  5. \(\cos(2x+y)=5x\)
  6. \(x^2y+y^2x=6\)
For an implicit equation, assume \(y\) is a differentiable function of \(x\).

Every time you differentiate a term containing \(y\), multiply by \(\frac{dy}{dx}\).
Solutions

  2. Implicit: \(x^2+y^2=25\), \(\ln(2xy^2)=4x\), \(\cos(2x+y)=5x\), \(x^2y+y^2x=6\).

Explicit vs implicit

Explicit

\[y=x+16\] \[y'=1\] The equation is already solved for \(y\).

Implicit

\[x^2+y^2=16\] Differentiate both sides: \[2x+2y\frac{dy}{dx}=0\] \[\frac{dy}{dx}=-\frac{x}{y}\]
Solutions
  1. \(2x+2yy'=0\Rightarrow y'=-\frac{x}{y}\).

The key idea

Chain rule inside implicit differentiation

\[\frac{d}{dx}[y]=\frac{dy}{dx}\] \[\frac{d}{dx}[y^n]=ny^{n-1}\frac{dy}{dx}\] \[\frac{d}{dx}[e^{5y}]=5e^{5y}\frac{dy}{dx}\] \[\frac{d}{dx}[\sin(2y)]=2\cos(2y)\frac{dy}{dx}\]
Differentiate with respect to \(x\), not with respect to the variable you see.
Solutions

  3. For tangent-line questions, substitute the point only after finding \(y'\), then use \(y-y_1=m(x-x_1)\).
  4. For horizontal tangents use numerator \(=0\) with denominator \(\ne0\); for vertical tangents use denominator \(=0\) with numerator \(\ne0\).

First example

Products and quotients inside implicit equations

Find \(\frac{dy}{dx}\).
  1. \(x^2y+y^2x=6\)
  2. \(\frac{x}{y}+\frac{y}{x}=3y\)
  3. \(3x^2-xy^2+y^{-1}=1\)
  4. \(5x^2+2x^2y+y^2=8\)
Before differentiating, decide:
  • Do I need product rule?
  • Can I rewrite with negative exponents?
  • Will \(y'\)-terms appear in more than one place?
Solutions
  1. \(x^2y+y^2x=6\Rightarrow y'=-\frac{2xy+y^2}{x^2+2xy}\).
  2. \(\frac{x}{y}+\frac{y}{x}=3y\Rightarrow y'=\frac{\frac{y}{x^2}-\frac1y}{-\frac{x}{y^2}+\frac1x-3}\).
  3. \(3x^2-xy^2+y^{-1}=1\Rightarrow y'=\frac{6x-y^2}{2xy+y^{-2}}\).
  4. \(5x^2+2x^2y+y^2=8\Rightarrow y'=-\frac{5x+2xy}{x^2+y}\).

Powers and radicals

Find \(\frac{dy}{dx}\).
  1. \(\sqrt{x}+\sqrt{y}=100\)
  2. \(\sqrt{x^2+y^2}=5\)
  3. \((2y^4+1)^2=5x^2\)
  4. \((x-y)^2-4x=20y\)
For radicals, rewrite first: \[\sqrt{y}=y^{1/2},\qquad \sqrt{x^2+y^2}=(x^2+y^2)^{1/2}.\] Then apply chain rule carefully.
Solutions

  2. \(\sqrt{x^2+y^2}=5\Rightarrow y'=-\frac{x}{y}\).

  4. \((x-y)^2-4x=20y\Rightarrow y'=\frac{x-y-2}{x-y+10}\).

Trigonometric implicit equations

Find \(\frac{dy}{dx}\).
  1. \(\sin(xy)=10x\)
  2. \(\cos(2x+y)=5x\)
  3. \(\sin x+2\cos(2y)=1\)
  4. \(\cos(xy)=x\)
  5. \(2x+\pi\cos y=2\pi\)
  6. \(y=\cos(xy)\)
Key derivatives: \[\frac{d}{dx}\sin(xy)=\cos(xy)(xy'+y)\] \[\frac{d}{dx}\cos(2x+y)=-\sin(2x+y)(2+y')\] \[\frac{d}{dx}\cos(2y)=-2\sin(2y)y'\]
Solutions
  1. \(\sin(xy)=10x\Rightarrow y'=\frac{10-y\cos(xy)}{x\cos(xy)}\).
  2. \(\cos(2x+y)=5x\Rightarrow y'=-\frac5{\sin(2x+y)}-2\).
  3. \(\sin x+2\cos(2y)=1\Rightarrow y'=\frac{\cos x}{4\sin(2y)}\).
  4. \(\cos(xy)=x\Rightarrow y'=\frac{-1-y\sin(xy)}{x\sin(xy)}\).
  5. \(2x+\pi\cos y=2\pi\Rightarrow y'=\frac{2}{\pi\sin y}\).
  6. \(y=\cos(xy)\Rightarrow y'=-\frac{y\sin(xy)}{1+x\sin(xy)}\).

Exponential implicit equations

Find \(\frac{dy}{dx}\).
  1. \(xe^y=2x+y^3\)
  2. \(2x-e^{4y^4}=y^2\)
  3. \(e^{xy}=x+y\)
  4. \(e^{x+y}=xy\)
Examples of inner derivatives: \[\frac{d}{dx}(e^y)=e^y y'\] \[\frac{d}{dx}(e^{4y^4})=16y^3e^{4y^4}y'\] \[\frac{d}{dx}(e^{xy})=e^{xy}(xy'+y)\]
Solutions
  1. \(xe^y=2x+y^3\Rightarrow y'=\frac{2-e^y}{xe^y-3y^2}\).
  2. \(2x-e^{4y^4}=y^2\Rightarrow y'=\frac{1}{y+8y^3e^{4y^4}}\).
  3. \(e^{xy}=x+y\Rightarrow y'=\frac{1-ye^{xy}}{xe^{xy}-1}\).
  4. \(e^{x+y}=xy\Rightarrow y'=\frac{y-e^{x+y}}{e^{x+y}-x}\).

Logarithmic implicit equations

Find \(\frac{dy}{dx}\).
  1. \(\ln(2xy^2)=4x\)
  2. \(\ln(xy)=\sin x\)
  3. \(\ln(x^2+y^2)=xy\)
  4. \(\ln y=x^2-y\)
Two possible approaches:
  • Differentiate directly using \(\frac{u'}{u}\).
  • Use log properties first if it makes the derivative cleaner.
Example: \[\ln(2xy^2)=\ln 2+\ln x+2\ln y.\]
Solutions
  1. \(\ln(2xy^2)=4x\Rightarrow y'=2y-\frac{y}{2x}\).

  4. \(\ln y=x^2-y\Rightarrow y'=\frac{2xy}{1+y}\).

Horizontal and vertical tangents

If \(y'=\frac{N(x,y)}{D(x,y)}\), then horizontal tangents usually occur where \(N=0\) and \(D\ne0\). Vertical tangents usually occur where \(D=0\) and \(N\ne0\).
Find all horizontal and vertical tangent lines.
  1. \(x^2+10x=-7y^5+10\)
  2. \(12x-3x^2=y^2-4y\)

  2. Given \(x^2+xy+y^2=7\), prove that \(\frac{d^2y}{dx^2}=-\frac{42}{(x+2y)^3}\).
  3. Given \(x^2+2xy+3y^2=18\), find \(\frac{d^2y}{dx^2}\) in terms of \(x\) and \(y\).
  4. Given \(x^2y+y^2x=6\), find \(\frac{d^2y}{dx^2}\) at \((1,2)\).
  1. Given \(x^3+y^3=3xy\), find \(\frac{d^2y}{dx^2}\) at \((1,2)\), if the point lies on the curve.
  2. Given \(y^2=x^3+3x^2\), find \(\frac{dy}{dx}\) and \(\frac{d^2y}{dx^2}\) at \((1,2)\), if the point lies on the curve.
  3. Given \(xy+\sin y=x^2\), find \(\frac{d^2y}{dx^2}\) at \((0,0)\).
  4. Given \(x^2+y^2+xy=3\), find all points where \(\frac{d^2y}{dx^2}=0\).
Solutions
  1. \(x^2+y^2=25\Rightarrow y''=-\frac{25}{y^3}\).
  2. \(x^2+xy+y^2=7\Rightarrow y''=-\frac{42}{(x+2y)^3}\).
  3. For \(x^2+2xy+3y^2=18\): \(y'=-\frac{x+y}{x+3y}\), then \(y''=-\frac{2+4y'+6(y')^2}{2x+6y}\).
  4. For \(x^2y+y^2x=6\) at \((1,2)\): \(y''=\frac{252}{125}\).
  5. \((1,2)\) is not on \(x^3+y^3=3xy\), so that question is invalid.
  6. For \(y^2=x^3+3x^2\) at \((1,2)\): \(y'=\frac94\), \(y''=\frac{15}{32}\).
  7. For \(xy+\sin y=x^2\) at \((0,0)\): \(y''=2\).

Challenge: tangent lines at the origin

  1. Given \(y^2=x^2(x+1)\), find the tangent lines at the origin.
  2. Given \(y^2=x^4+x^5\), find the tangent line at the origin.
  3. For \(x^2y^2=x^4+y^4\), show that \((0,0)\) is singular. Determine whether any real tangent lines pass through the origin.
  4. For \(x^3+y^3=3xy\), find the possible tangent lines at \((0,0)\).
  1. For \(y^2=x^3\), find the tangent line at the origin.
  2. For \(x^4+y^4=2x^2y^2\), find all possible tangent lines at the origin.
  3. For \((x^2+y^2)^2=4x^2y\), find all tangent lines at the origin.
  4. For \(y^3=x^2(x+1)\), find the tangent line at the origin.
Solutions
  1. \(y^2=x^2(x+1)\): \(y=x\) and \(y=-x\).
  2. \(y^2=x^4+x^5\): \(y=0\).
  3. \(x^2y^2=x^4+y^4\): no real tangent slopes.
  4. \(x^3+y^3=3xy\): \(x=0\) and \(y=0\).
  5. \(y^2=x^3\): \(y=0\).
  6. \(x^4+y^4=2x^2y^2\): \(y=x\), \(y=-x\).
  7. \((x^2+y^2)^2=4x^2y\): possible tangent lines \(x=0\), \(y=0\).
  8. \(y^3=x^2(x+1)\): vertical tangent \(x=0\).

Challenge: singular tangent lines

  1. If \(x^2+y^2+x^3-y^3=0\), find all possible tangent lines at the origin.
  2. If \(x^4+y^4=x^2y^2+x^5\), find all possible tangent lines at the origin.
  3. If \(y^4=x^6+x^7\), find the tangent line at the origin.
  1. A curve is defined by \(x^4+y^4=x^2y^2+x^5\). Explain why direct implicit differentiation at \((0,0)\) does not immediately give the tangent line. Let \(y=mx\), find all possible slopes, and write the tangent lines.
Solutions
  1. \(x^2+y^2+x^3-y^3=0\): no real linear tangent from the lowest-degree equation.
  2. \(x^4+y^4=x^2y^2+x^5\): no real tangent slopes because \(m^4-m^2+1=0\).
  3. \(y^4=x^6+x^7\): tangent line \(y=0\).
  4. At a singular origin, direct implicit differentiation usually gives \(0=0\); use lowest-degree terms with \(y=mx\).

  1. Find \(\frac{dy}{dx}\).
  2. Show that \(\frac{d^2y}{dx^2}=-\frac{42}{(x+2y)^3}\).
  3. Determine whether the curve is concave up or down at \((1,2)\), if the point lies on the curve.
Solutions
  1. For \(x^3+y^3=6xy\): \(y'=\frac{2y-x^2}{y^2-2x}\). At \((3,3)\), tangent \(y-3=-(x-3)\).

  2. Find \(\frac{dy}{dx}\) when \(xy\lt0\).
