4.5

Use the tangent line as a local model for the curve.

Core idea

Near a point of tangency, the graph and tangent line are almost indistinguishable.

Definition

The formula

Linearization at x=a

\[L(x)=f(a)+f'(a)(x-a)\]
Use this when x is close to a. The input a is the center; x is the nearby target value.
Textbook connection

Differentials and increments

Actual change

\[\Delta y=f(a+\Delta x)-f(a)\]

Linear change

\[dy=f'(a)\,dx,\qquad dx=\Delta x\]
\[f(a+\Delta x)\approx f(a)+f'(a)\Delta x\]

Concavity rule

Overestimate or underestimate?

Concave up

The curve lies above its tangent line, so the tangent-line value is an underestimate.

Concave down

The curve lies below its tangent line, so the tangent-line value is an overestimate.
The rule is local: it must be true on the interval between a and the target x.

Method

Procedure

  1. Choose a nearby center \(a\) where \(f(a)\) and \(f'(a)\) are known.
  2. Write \(L(x)=f(a)+f'(a)(x-a)\).
  3. Substitute the target input.
  4. Use concavity to decide under or over.
  5. If exact value is available, calculate \(|f(x)-L(x)|\).
