4.6 L’Hôpital’s Rule

4.6

Use derivatives to evaluate selected indeterminate limits.

Central idea

When a quotient approaches \(0/0\) or \(\infty/\infty\), compare the rates at which numerator and denominator approach their limiting values.
0/0∞/∞L’Hôpital’s RuleRepeated useWhen not to use itAP notation
Lesson overview

Learning targets

Identify

Check direct substitution before using any rule.

Apply

Use L’Hôpital’s Rule only for \(0/0\) or \(\infty/\infty\).

Repeat

Apply the rule again only if the new quotient is still indeterminate.

Justify

Write AP-style limit notation and explain when the rule does not apply.
Exam scope

AP boundary for this lesson

Assessed forms

\[\frac{0}{0}\qquad\text{and}\qquad \frac{\infty}{\infty}\]
Other indeterminate forms such as \(\infty-\infty\), \(0\cdot\infty\), \(1^\infty\), \(0^0\), and \(\infty^0\) exist, but those conversions are not required for AP Calculus AB/BC assessment in this lesson.
Decision step

First check: direct substitution

Defined number

If substitution gives a real number, stop.

Not indeterminate

If substitution gives nonzero over zero, analyze one-sided behavior; do not use L’Hôpital.

Indeterminate quotient

Only \(0/0\) or \(\infty/\infty\) can trigger L’Hôpital’s Rule.
Formal statement

The rule

L’Hôpital’s Rule

If \(\displaystyle \lim_{x\to a}f(x)=\lim_{x\to a}g(x)=0\) or both limits are infinite, and the derivative quotient has a limit, then \[\lim_{x\to a}\frac{f(x)}{g(x)}=\lim_{x\to a}\frac{f'(x)}{g'(x)}.\]
The rule also works for one-sided limits and limits as \(x\to\pm\infty\), provided the hypotheses are satisfied.
Communication

AP notation protocol

  1. Write the original limit.
  2. Show the numerator and denominator approach \(0\) or both approach \(\pm\infty\).
  3. State the form: \(0/0\) or \(\infty/\infty\).
  4. Write “By L’Hôpital’s Rule” and replace the quotient by \(f'(x)/g'(x)\).
  5. Evaluate the new limit. Repeat only if the new quotient is still indeterminate.
Critical warning

L’Hôpital is not the quotient rule

Limit problem

\[\lim_{x\to a}\frac{f(x)}{g(x)}\]For \(0/0\) or \(\infty/\infty\), compare derivative quotient \(f'(x)/g'(x)\).

Derivative problem

\[\frac{d}{dx}\left(\frac{f(x)}{g(x)}\right)\]Use the quotient rule, not L’Hôpital’s Rule.

Error analysis

Textbook caution: do not overapply

Find the mistake: \(\displaystyle \lim_{x\to0}\frac{x^2}{e^x-1}=\lim_{x\to0}\frac{2x}{e^x}=\lim_{x\to0}\frac{2}{e^x}=2\).

Wrong second application

The slider moves the input and updates the graph, quotient value, and derivative quotient value.
Show solution
The first L’Hôpital step is valid because the original form is \(0/0\). After one step, \[\lim_{x\to0}\frac{2x}{e^x}=0,\] which is no longer indeterminate. Applying L’Hôpital a second time is invalid. The correct limit is \(0\).

Textbook one-sided example

Evaluate \(\displaystyle \lim_{x\to0^+}\frac{\ln x}{\csc x}\).

One-sided ∞/∞ with simplification

The slider moves the input and updates the graph, quotient value, and derivative quotient value.
Show solution
The form is \(\infty/\infty\) in magnitude. One L’Hôpital step gives \[\lim_{x\to0^+}\frac{1/x}{-\csc x\cot x}.\] Simplify to \(-\frac{\sin x\tan x}{x}\), which tends to \(0\).
Local linearity connection

Why the derivative quotient makes sense

If \(f(c)=g(c)=0\), then near \(x=c\): \[f(x)\approx f'(c)(x-c),\qquad g(x)\approx g'(c)(x-c).\] So \[\frac{f(x)}{g(x)}\approx\frac{f'(c)}{g'(c)}.\]
This connects L’Hôpital’s Rule to the linear approximation lesson.

Show solution

Before using the rule

Common mistakes checklist

Do not

  • Use L’Hôpital on a derivative problem.
  • Use it on a nonzero-over-zero form.
  • Apply it twice if the first derivative quotient is no longer indeterminate.
  • Drop the limit notation before the final value.

Do

  • Substitute first.
  • State \(0/0\) or \(\infty/\infty\).
  • Differentiate numerator and denominator separately.
  • Simplify if that is easier than repeated use.
