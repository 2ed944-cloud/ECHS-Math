2.4 Basic Derivative Rules

Basic Derivative Rules

Use derivative rules to differentiate efficiently, connect rules to limits, find tangent and normal lines, work with higher-order derivatives, and handle AP-style interpretation questions.

\(f'(x)=\displaystyle\lim_{h\to0}\frac{f(x+h)-f(x)}{h}\)
Every rule in this lesson comes from this limit.

2. Estimate the slope of \(y=x^2\) at \(x=2\). Then compare it with the slope from \[ \lim_{h\to0}\frac{(2+h)^2-2^2}{h}. \]

AP Calculus AB

What a derivative rule does

A derivative rule is a shortcut for a limit that has already been proven.
Limit form
\(\displaystyle f'(x)=\lim_{h\to0}\frac{f(x+h)-f(x)}{h}\)
Rule form
\(\displaystyle \frac{d}{dx}\left[x^n\right]=nx^{n-1}\)
Meaning
The derivative gives the slope of the tangent line and the instantaneous rate of change.
Student task: For each rule today, explain what limit it replaces.

Functional equation and derivative I

Assume \(h(x+y)=h(x)h(y)\) for all real \(x,y\).

1. If \(h(x)=1+xL(x)\) and \(\lim_{x\to0}L(x)=2\), prove that \(h\) is differentiable on \(\mathbb{R}\), and find \(h'(x)\) in terms of \(h(x)\).
2. If \(\displaystyle\lim_{x\to0}\frac{h(x)-1}{x}=5\), prove that \(h\) is differentiable on \(\mathbb{R}\), and prove \(h'(x)=5h(x)\).
3. If \(h\) is differentiable at \(0\) and \(h'(0)=k\), prove that \(h\) is differentiable for every real \(x\), and prove \(h'(x)=kh(x)\).
4. If \(h(0)=1\) and \(\displaystyle\lim_{x\to0}\frac{h(x)-1}{x}=3\), prove that \[\lim_{t\to0}\frac{h(x+t)-h(x)}{t}=3h(x).\]
Use \(h(x+t)-h(x)=h(x)(h(t)-1)\). Therefore \(\frac{h(x+t)-h(x)}{t}=h(x)\frac{h(t)-1}{t}\). The coefficient is the derivative at \(0\). Thus the answers are \(h'(x)=2h(x)\), \(h'(x)=5h(x)\), \(h'(x)=kh(x)\), and \(3h(x)\).

Functional equation and derivative II

Assume \(h(x+y)=h(x)h(y)\) for all real \(x,y\).

5. If \(\displaystyle\lim_{x\to0}\frac{h(x)-1}{x}=4\), prove that \(h(x)>0\) for all real \(x\), and then prove \(h'(x)=4h(x)\).
6. If \(h(x)=1+xL(x)\) and \(\lim_{x\to0}L(x)=1\), define \(F(x)=h(x^2)\). Prove that \(F\) is differentiable for every real \(x\), and find \(F'(x)\).
7. If \(h(x)=1+xL(x)\) and \(\lim_{x\to0}L(x)=1\), first prove \(h(x)>0\). Then define \(g(x)=\ln(h(x))\). Prove that \(g\) is differentiable and find \(g'(x)\).
8. If \(h(x)=1+xL(x)\) and \(\lim_{x\to0}L(x)=1\), prove \(h(-x)=\frac{1}{h(x)}\) and use it to show \(h(x)\ne0\) for all real \(x\).
5. Since \(h(x)=h(x/2)^2\) and \(h(x)h(-x)=h(0)=1\), we get \(h(x)>0\). Then \(h'(x)=4h(x)\). 6. Since \(h'(u)=h(u)\), \(F'(x)=2x h(x^2)\). 7. \(g'(x)=\frac{h'(x)}{h(x)}=1\). 8. \(h(x)h(-x)=h(0)=1\), so \(h(-x)=1/h(x)\) and \(h(x)\ne0\).

Functional equation and derivative III

9. Let \(h(x+y)=h(x)h(y)\) for all real \(x,y\). Suppose \(h(x)=1+xL(x)\) and \(\lim_{x\to0}L(x)=1\). Prove that \[\lim_{x\to0}\frac{h(ax)-1}{x}=a\] for every real constant \(a\).
Write \(\frac{h(ax)-1}{x}=a\cdot\frac{h(ax)-1}{ax}\) when \(a\ne0\). As \(x\to0\), \(ax\to0\), so the limit is \(a\). If \(a=0\), the numerator is \(h(0)-1=0\), so the limit is also \(0\).

One-sided derivative limits I

Assume \(h(3)=6\), \(h'_+(3)=4\), and \(h'_-(3)=2\).

1. Determine whether each limit exists. If it exists, find its value. \[\lim_{x\to3}\frac{h(x)-6}{|x-3|}\qquad\text{and}\qquad \lim_{x\to3}\frac{|h(x)-6|}{x-3}.\]
2. Find \(a\), if it exists, so that \[\lim_{x\to3}\frac{h(x)-6-a|x-3|}{x-3}\] exists.
3. Find \[\lim_{x\to3}\frac{h(x)-6}{\sqrt{|x-3|}}.\]
4. Determine whether \[\lim_{x\to3}\frac{h(x)-h(3)}{x-3+|x-3|}\] exists.
Use \(h(x)-6\sim4(x-3)\) from the right and \(h(x)-6\sim2(x-3)\) from the left. 1. Both limits do not exist: right values are \(4\), left values are \(-2\). 2. Right value \(4-a\), left value \(2+a\), so \(a=1\) and the limit is \(3\). 3. The limit is \(0\). 4. It does not exist because the denominator is \(0\) for \(x<3\).

1. If \(f\) is differentiable at \(x=1\) and \[\lim_{h\to0}\frac{f(1+h)-3h+h^2}{h}=8,\] find \(f(1)+f'(1)\).
2. If \(f\) is differentiable at \(x=1\) and \[\lim_{h\to0}\frac{f(1+h)-f(1-h)+4h}{2h}=7,\] find \(f'(1)\).
3. If \(f\) is differentiable at \(x=1\) and \[\lim_{h\to0}\frac{f(1+h)+ah}{h}=b,\] find \(f(1)\) and \(f'(1)\) in terms of \(a\) and \(b\).
4. Let \(f\) be differentiable at \(x=2\). If \[\lim_{h\to0}\frac{f(2+h)-7}{h}=4,\] find \(f(2)\) and \(f'(2)\).
1. Finite limit forces \(f(1)=0\), and \(f'(1)-3=8\), so \(f(1)+f'(1)=11\). 2. \(f'(1)+2=7\Rightarrow f'(1)=5\). 3. Finite limit forces \(f(1)=0\), and \(f'(1)+a=b\), so \(f'(1)=b-a\). 4. \(f(2)=7\) and \(f'(2)=4\).

Derivative definition challenge IV

13. Let \(f\) be differentiable at \(x=1\). For constants \(a\) and \(\lambda\), define \[L(a,\lambda)=\lim_{h\to0}\frac{f(1+h+a|h|)+\lambda f(1-h+a|h|)-(1+\lambda)f(1)}{h}.\] Find all values of \(a\) and \(\lambda\) for which \(L(a,\lambda)\) is guaranteed to exist for every differentiable \(f\), and express the limit in terms of \(f'(1)\).
14. Construct a function \(f\) such that \(f\) is differentiable at \(0\), \(f'(0)=0\), but \(f''(0)\) does not exist. Justify using limits.
13. The right coefficient is \(1+a+\lambda(a-1)\). The left coefficient is \(1-a-\lambda(1+a)\). Equality requires \(a(1+\lambda)=0\). If \(a=0\), then \(L=(1-\lambda)f'(1)\). If \(\lambda=-1\), then \(L=2f'(1)\). 14. One example is \(f(x)=x^2\sin(1/x)\) for \(x\ne0\), and \(f(0)=0\). Then \(f'(0)=\lim_{x\to0}x\sin(1/x)=0\), but \(f''(0)\) does not exist.

Line tangent to another line

Find all values of \(k\) such that the graph of \[ f(x)=x^2+kx \] is tangent to the line \[ y=2x-9. \]
At tangency \(x=t\), slopes match: \(2t+k=2\Rightarrow k=2-2t\). Also the point lies on the line: \(t^2+kt=2t-9\). Substitute \(k\): \(t^2+(2-2t)t=2t-9\Rightarrow t^2=9\). Thus \(t=3\) gives \(k=-4\), and \(t=-3\) gives \(k=8\).
