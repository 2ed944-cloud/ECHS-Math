Big idea

A limit describes the value a function approaches, not necessarily the value the function has.

\[\lim_{x\to c} f(x)=L\]

Key sentence

A limit asks: “What happens to \(f(x)\) as \(x\) gets close to \(c\)?”

\[x\to c \quad \Longrightarrow \quad f(x)\to L\]
Limit notation

Three types of limits

Right-hand limit

\[\lim_{x\to c^+} f(x)=L\]

Approach \(c\) from values greater than \(c\).

Left-hand limit

\[\lim_{x\to c^-} f(x)=L\]

Approach \(c\) from values less than \(c\).

General limit

\[\lim_{x\to c} f(x)=L\]

Both one-sided limits approach the same value.

Activity: Write a verbal explanation for each notation without using the word “equals” too early. Use “approaches.”

Concept check

The limit and the function value are different questions

Limit question

\[\lim_{x\to c} f(x)\]

What y-value does the graph approach as \(x\) gets close to \(c\)?

Function value question

\[f(c)\]

What y-value is assigned exactly at \(x=c\)?

Graphical interpretation

The “highway” view of one-sided limits

Walk along the graph

If the graph is a road and you travel toward a vertical line \(x=c\):

  • From the right means your x-values are greater than \(c\).
  • From the left means your x-values are less than \(c\).
  • The limit is the y-value you approach.
\[\lim_{x\to c^-} f(x) \quad \text{and} \quad \lim_{x\to c^+} f(x)\]

Limit Existence Theorem

When does a two-sided limit exist?

The general limit exists exactly when the left-hand and right-hand limits exist and are equal.

\[\lim_{x\to c} f(x)=L \quad \Longleftrightarrow \quad \lim_{x\to c^-} f(x)=L \text{ and } \lim_{x\to c^+} f(x)=L\]
Justification frame: “The limit \(\lim_{x\to c}f(x)\) ______ because \(\lim_{x\to c^-}f(x)=\) ____ and \(\lim_{x\to c^+}f(x)=\) ____.”
Quick check

Does the limit exist?

Level 1

If \(\lim_{x\to 2^-}f(x)=5\) and \(\lim_{x\to 2^+}f(x)=5\), then \(\lim_{x\to 2}f(x)=\) ?

\(5\). The one-sided limits are equal.
Level 1

If \(\lim_{x\to -1^-}g(x)=3\) and \(\lim_{x\to -1^+}g(x)=7\), does \(\lim_{x\to -1}g(x)\) exist?

No. The one-sided limits are different.
Level 2

If \(\lim_{x\to 4}h(x)=9\), what must be true about the one-sided limits?

Both must exist and equal 9.
Level 2

Can \(f(2)=100\) while \(\lim_{x\to 2} f(x)=6\)?

Yes. A function value at the point can differ from the limiting value.
Limits can fail to exist

Three common DNE situations

1. Jump

The left-hand and right-hand limits are finite but unequal.

2. Infinite behavior

The function grows without bound near the x-value.

3. Oscillation

The function does not approach one y-value.

Student activity

Justify why the limit does not exist

Graph A

A graph approaches 2 from the left and 5 from the right as \(x\to 1\).

The limit DNE because \(\lim_{x\to1^-}f(x)=2\) and \(\lim_{x\to1^+}f(x)=5\), and \(2\ne5\).
Graph B

A graph has a vertical asymptote at \(x=-3\), with both sides increasing without bound.

The finite limit DNE because \(f(x)\to \infty\) as \(x\to -3\). It does not approach a real number.
Estimating from tables

Use values near the point, not only the point

Function with a hole

\[g(x)=\frac{x^2-9}{x-3}\]

Direct substitution at \(x=3\) gives \(\dfrac{0}{0}\), but nearby values reveal the limit.

From table to notation

Write the conclusion using correct notation

From the table and graph for \(g(x)=\dfrac{x^2-9}{x-3}\), complete:

\[\lim_{x\to3^-}g(x)=\ ?\]
\[\lim_{x\to3^+}g(x)=\ ?\]
\[\lim_{x\to3}g(x)=\ ?\]
\(\lim_{x\to3^-}g(x)=6\), \(\lim_{x\to3^+}g(x)=6\), so \(\lim_{x\to3}g(x)=6\). The value \(g(3)\) is undefined, but the limit exists.
Interpret analytic notation

Read each notation as a sentence

Basic

\(\lim_{x\to 6} f(x)=10\)

As \(x\) approaches 6 from both sides, \(f(x)\) approaches 10.
Intermediate

\(\lim_{x\to -4^-} f(x)=\infty\)

As \(x\) approaches \(-4\) from the left, \(f(x)\) increases without bound.
Intermediate

\(\lim_{x\to 0^+} f(x)=-\infty\)

As \(x\) approaches 0 from the right, \(f(x)\) decreases without bound.
Challenge

\(\lim_{x\to 2^-}f(x)=4\), \(\lim_{x\to 2^+}f(x)=4\), and \(f(2)=-1\).

\(\lim_{x\to2}f(x)=4\), even though the actual function value is \(-1\).
Reasoning with properties

Properties help justify claims about limits

When substitution is valid

For many familiar continuous functions, the limit can be found by substitution.

\[\lim_{x\to c}p(x)=p(c)\]

This is true for polynomial functions and many continuous functions.

When substitution is not enough

For a rational function, direct substitution may produce \(\dfrac{0}{0}\) or reveal an asymptote.

\[\frac{0}{0}\text{ is indeterminate, not an answer.}\]
Student activity

Choose the best first strategy

Level 1

\(\lim_{x\to 2}(3x^2-5x+1)\)

Substitute: \(3(2)^2-5(2)+1=3\).
Level 2

\(\lim_{x\to 3}\dfrac{x^2-9}{x-3}\)

Factor and simplify: \(\dfrac{(x-3)(x+3)}{x-3}=x+3\), so the limit is 6.
Level 2

\(\lim_{x\to -2}\dfrac{x+2}{x^2-4}\)

\(\dfrac{x+2}{(x-2)(x+2)}=\dfrac{1}{x-2}\), so the limit is \(-\dfrac{1}{4}\).
Level 3

\(\lim_{x\to 1}\dfrac{|x-1|}{x-1}\)

Left-hand limit is \(-1\), right-hand limit is \(1\), so the two-sided limit DNE.

Level 3 and Level 4 practice

Level 3

Evaluate and justify: \(\lim_{x\to 4}\dfrac{x^2-16}{x-4}\).

Factor: \(x^2-16=(x-4)(x+4)\). For \(x\ne4\), the expression equals \(x+4\), so the limit is 8.
Level 3

Evaluate and justify: \(\lim_{x\to 0}\dfrac{|x|}{x}\).

Left-hand limit is \(-1\); right-hand limit is \(1\). Since they are not equal, the two-sided limit DNE.

Level 4

Create a piecewise function where \(\lim_{x\to1^-}f(x)=2\) and \(\lim_{x\to1^+}f(x)=7\).

One example: \(f(x)=2\) for \(x<1\), and \(f(x)=7\) for \(x\ge1\). The two-sided limit DNE.
AP-style practice

Short constructed response

A function \(f\) has the following behavior near \(x=2\):

StatementValue
\(\lim_{x\to2^-}f(x)\)\(4\)
\(\lim_{x\to2^+}f(x)\)\(4\)
\(f(2)\)undefined

Task: Determine whether \(\lim_{x\to2}f(x)\) exists. If it exists, give its value. Justify your answer.

The limit exists and equals \(4\). Since \(\lim_{x\to2^-}f(x)=4\) and \(\lim_{x\to2^+}f(x)=4\), the one-sided limits are equal. Therefore, by the Limit Existence Theorem, \(\lim_{x\to2}f(x)=4\). The fact that \(f(2)\) is undefined does not prevent the limit from existing.

Lesson summary

What matters most in Topic 1.2?

Limits describe approaching behavior

A limit is about what \(f(x)\) approaches as \(x\) approaches a value.

One-sided limits control existence

The two-sided limit exists only when the left-hand and right-hand limits agree.

Notation matters

Use \(x\to c^-\), \(x\to c^+\), and \(x\to c\) accurately.

Graphs and tables give evidence

Estimate limits from nearby values and graphical behavior.
