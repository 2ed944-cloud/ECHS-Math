# AP Calculus AB Unit 1 - Separated Delivery Package

## Website-ready student files

Upload the contents of `website-ready/` to the root of the ECHS-Math repository. This preserves the recommended path:

`worksheets/ap-calculus/unit-1/`

The folder contains 16 individual student worksheets, one for every official Unit 1 topic.

## Teacher files

`teacher-answer-keys/` contains 16 separate answer keys. Keep these outside the public website unless student access is intended.

## Editable sources

`editable-latex/student/` and `editable-latex/teacher/` contain one standalone LaTeX source for each topic.

## Integration files

The `integration/` folder contains:

- the complete official Unit 1 topic-name list;
- website filenames and full public URLs;
- copy-ready resource strings for `data/courses.js`;
- a one-line `portal.js` patch so more than two resource chips can be displayed.

## Naming standard

Student PDFs use `worksheet-1-x-full-official-topic-name.pdf`. Teacher files use `answer-key-1-x-full-official-topic-name.pdf`. All filenames are lowercase, URL-safe, and hyphenated.
